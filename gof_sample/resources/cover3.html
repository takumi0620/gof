Java サンプルコードについて

この CD-ROM は、23 個全てのパターンについての Java 版のサンプルコードを収録しています。このサンプルコードは、本書に掲載されている C++ 版サンプルコードを Java で書き直したものです。その大半は C++ 版からの直接的な翻訳ですが、一部は Java の事情に合わせてその形を変えています。C++ 版で示される例からは逸脱せずに、なおかつ Java 1.1 言語および JDK 1.1 のプログラミングスタイルから外れることのないよう留意しています。Java は、その言語仕様と共にクラスライブラリを提供しています。クラスライブラリが本書に掲載されるデザインパターンに適したインターフェースを提供している場合は、サンプルコードではそれを使用しました。例えば、Iterator パターンでは iterator クラスとして java.util.Enumeration を使用したサンプルコードを用意しました。また、Java は具象クラスの多重継承を許しません。そうした事情に合わせ、例えば Adapter パターンでは C++ 版で示されるクラスアダプタの代わりに、オブジェクトアダプタの例を示しました。

本書では C++ 版サンプルコードで頻繁に用いられるユーティリティクラスを foundation classes として付録に掲載していますが、Java 版ではこうしたユーティリティクラスには JDK 1.1 が提供する Java 標準のクラスを使用しています。例えば C++ 版で使用する List クラスの代わりに、java.util.Vector クラスを用いています。




Java サンプルコードの表記

Java サンプルコードに現れるクラスの定義の冒頭で「役割」を示しています。その表記は次の形式に従っています：

　役割：<役割名 1><役割名 2>/パターン名, ...

各々の要素の意味は次の通りです：

　<役割名 1>：「構成要素」節で定義されるクラス（役割）の名前
　<役割名 2>：クラスが複数の役割を担っている場合続けて書きます
　パターン名：役割を定義しているパターンの名前

クラスが複数のパターンの構成要素である場合、カンマで区切って該当する全てのパターンそれぞれにおける役割を示します。

サンプルコード中では所々に import 文が挿入されていますが、その効果はそれが挿入されている行以下そのサンプルコードの最後までの間で有効であるものとします。
