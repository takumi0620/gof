この CD-ROM について

この CD-ROM は、本書「デザインパターン」の「デザインパターンカタログ」から全てのパターンの記述を抜粋して収録したものです。HTML 形式のコンテンツは、23 個のデザインパターンの記述をその名前によって検索できるようになっています。本書と併せて、アクセス性に優れた CD-ROM 版のデザインパターンカタログをご活用ください。

この CD-ROM では特別に Java 1.1 で書かれたサンプルコードを用意しました。それらは Java プログラミングにおけるデザインパターン適用のより直接的な例となるように配慮されています。また、本書及び本 CD-ROM に収録された C++ 版サンプルコードと比較することにより、両言語に於けるデザインパターン適用の相似点や相違点を理解する手助けが得られます。

この CD-ROM が、本書「デザインパターン」読者の方々によるデザインパターンの理解と活用のお役に立つことを願います。
